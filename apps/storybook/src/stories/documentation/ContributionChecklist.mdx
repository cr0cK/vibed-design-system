# Contribution Checklist

## Before opening PR

1. Component follows atom/molecule/organism placement.
2. Props interface is explicit and complete.
3. No prop destructuring in function signature.
4. Styling uses `styled.*` + `buildVariants(props)`.
5. Variant values use explicit string unions.
6. If component extends `HTMLAttributes`, pass supported DOM props through to the root element; otherwise do not extend it.
7. Overlay-like behavior (`outside click`, `focus trap`, `portal`) reuses utility components unless a clear reason is documented.
8. Shared state uses context or small Zustand slices where needed.
9. Storybook story added or updated when component behavior changes.
10. `pnpm typecheck` passes.
11. `pnpm build` passes.
12. `pnpm --filter @vibed/storybook build` passes.
13. `pnpm test:e2e` passes (Playwright against built Storybook).
14. If interaction behavior changed, add or update focused interaction/a11y tests.
